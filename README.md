### Overview
EasySpeakBot is an AI-driven bot specifically crafted for Russian speakers who want to master English through active conversation. The project bridges the gap between passive learning and confident speaking.

### Core Features
- **Contextual Conversation:** Engage in natural dialogues on any topic.
- **Instant Feedback:** The bot detects typical mistakes made by Russian speakers (articles, tenses, word order) and provides corrections.
- **Voice-Ready (In Progress):** A roadmap feature to allow full voice immersion with transcription to help track pronunciation and fluency.

- **AI Model:** GPT-4o / Claude (or specify your choice)
- **Speech Tech:** OpenAI Whisper / Google STT (planned)
